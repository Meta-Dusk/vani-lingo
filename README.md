# VaniLingo

This is a simple Chinese study app made with `Flet`. I've made this as a gift for someone.
The name __"VaniLingo"__ comes from the name of that certain someone (*Vani*) and then combining
it with the suffix of *DuoLingo*.

## 📃 Feature List

| Feature | Description |
| ------- | ----------- |
| __HSK 1 Data__ | HSK 1 (*Hanyu Shuiping Kaoshi*) is used for the data displayed. |
| __Generate Examples__ | If connected to an internet connection, the app can generate example sentences that uses the shown kanji. |
| __Text-to-Speech__ | Audio playbacks on the pronounciations of the kanji displayed are available. |
