# VaniLingo

This is a simple Chinese study app made with `Flet`. I've made this as a gift for someone.
The name __"VaniLingo"__ comes from the name of that certain someone (*Vani*) and then combining
it with the suffix of *DuoLingo*.

## 📃 Feature List

| Feature | Description |
| ------- | ----------- |
| __HSK 1-3 Data__ | HSK 1-3 (*Hanyu Shuiping Kaoshi*) is used for the data displayed. |
| __Generate Examples__ | The app can generate example sentences that uses the shown kanji (*Online only*). |
| __Text-to-Speech__ | Audio playbacks on the pronounciations of the kanji displayed are available (*Online only*). |
| __HSK Level Selection__ | User can select which level of HSK they want. |
| __Offline Mode Support__ | The app can operate with minimal features on no WiFi connection. |
